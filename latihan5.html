Apa itu css

Css atau cascading style sheet adalah dokumen web yang memiliki fungsi yang mengatur tampilan dokumen lain yaitu HTML. beberapa contoh penmanfaatan css pada html adalah untuk mengatur warna latar belakang haya huruf jarak anatr element dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadanya tidak mengggantikan html melainkan sebagai pelengkap agar tampilan dokumen html menjadi lebih indah dan dinamis BERIKUT INI ADALAH CONNTOH DOKUMEN CSS.

Bagaimana cara menggunakan css

ada 3 cara menggunakan html

  • inline styles:menuliskan code css langsung pada tag yangn akan di ubah dan di atur style css
  • internal/embedded styles: menuliskan code css pada bagian header dokumen html
  • Eksternal/linked styles:menuliskan code css pada file yang terpisah dan di panggil melalui tag/link dari dokumen html
